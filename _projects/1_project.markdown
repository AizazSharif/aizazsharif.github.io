This is part of `Graph Theory` course for C lanuage implementation of finding Maximal and Maximum Clique Using Bron Kerbosch Algorithm

Example output For a Graph

G.V = { 1, 2, 3, 4, 5 } G.E = { 1->2, 1->3, 2->3, 2->4, 3->5, 4->5 }

* Maximal Clique : C.V = { 1, 2, 3 } { 2, 4 } { 3, 5 } { 4, 5 }
* Maximum: C.V = { 1, 2, 3 }

You can find the project up and running details along with the code [here](https://github.com/AizazSharif/Bron-Kerbosch-Algorithm). This project is a small contribution to the C based Graph library under the supervision of [Dr. Omar Usman Khan](https://pwr.nu.edu.pk/profile/omar.khan/).
